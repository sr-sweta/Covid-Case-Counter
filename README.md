# Covid-Case-Counter

It gives the current count of covid cases. The counts are coming throught an API call. Its bbackend is made using Django default SQlite3 database.

## Stacks used

- Django
- Python
- Rest API
- JQuery
- HTML
- CSS
- Bootstrap
- Javascript
- JSON
